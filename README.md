KAYAK Promotional Email 

A responsive promotional email inspired by the design and marketing style of travel platforms like KAYAK. This project recreates a modern travel promotion email using clean HTML/CSS, responsive layout techniques, and marketing-focused design patterns commonly used in real email campaigns.

The purpose of this project is to demonstrate how developers and digital marketers can build conversion-focused promotional emails that are optimized for multiple devices and email clients.
